Setup Virtual Environment
```
python3 -m venv venv
```

Install Packages
```
pip install discord.py
pip install django
pip install djangorestframework
pip install requests
pip install asyncio
```

Run Server (cd to main folder)
```
python3 manage.py runserver
```
